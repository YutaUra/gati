# File Search

## What

Add incremental file name search (`/` key) to the file tree. Filters entries as user types, shows matching files with ancestor directories, supports j/k navigation, Enter to confirm, Escape to cancel.

## Why

In large projects, scrolling and expanding directories to find a specific file is slow. Search provides immediate access to any file by name.

## Capabilities

- **Modified capability: file-tree** — Search mode with incremental filtering, search input UI
